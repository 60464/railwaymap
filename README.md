# railwaymap
介绍 Introduction  
本软件是基于Pyehcart库，对中国各铁路局管辖范围进行可视化展示。各路局地图数据是行政地图和相关车站编辑生成的json数据。  
本库全部功能都是基于Pyechart，最终文件可以生成html文件或者图片文件，也可以进行相关修改后使用echart在数据大屏中生成动态地图。  
This software is based on Pyehcart library to visually display the jurisdiction of each railway bureau in China. The map data of each railway bureau is the json data generated by the editing of administrative maps and related stations.  
All functions of this library are based on Pyechart, the final file can generate html files or image files, you can also make relevant modifications to use echart to generate dynamic maps in the data screen.

文件说明 File specification  
json_data.py 包括所有路局的json数据  
railway_map.py 包括所有地图绘制函数   
example.py 包括三个基本示例函数  
json_data.py contains json data for all bureaus  
railway_map.py includes all mapping functions  
Example.py includes three basic example functions  

安装 Installation  
pip install railwaymap  

例子 Example  
from railwaymap.example import normal_map_example, heat_map_example, single_bureau_map_example  
#生成各路局总体地图  
normal_map_example()   
#生成各路局总体热力地图  
heat_map_example()  
#生成上海局内车站坐标位置图  
single_bureau_map_example()
